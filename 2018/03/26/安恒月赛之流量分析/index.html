流量分析题目内容

题目一

wireshark打开流量包，首先过滤出http协议，一般web应用比较容易分析。
大体浏览一下，发现X-Forward-For和wvs，意思指纹。

上网查了一下wvs指纹。
wvs指纹：http://www.freebuf.com/articles/web/16806.html
查看了一下资料，了解到wvs的一些标识，其中包括print(md5….),恰好我们刚才看到过。
我们根据特征筛选一下特殊标志，http contains “print(md5(“

发现好多的数据包，这说明黑客通过wvs扫描公司发送的数据包，所以黑客的ip地址就是183.129.152.140啦

题目二

首先分析一下题目，是要找到黑客发送钓鱼邮件所使用的的邮箱，之前在前面看到过webmail，还是首先筛选http，邮箱是在118.194.196.232:8090这台服务器上运行。

然后我们根据服务器地址继续筛选一下http contains “118.194.196.232”，首先是Referer中包含的webmali/login9.php,应该是一个登陆界面，cookie中存在着一些登陆信息

既然是邮件发送，我们可以想到SMTP和pop3协议，随机我们筛选一下数据(有些数据包中没有，可以去其他的数据包中找找)，可以看到mail from xsser@live.cn，这就是黑客的邮箱

题目三

我们可以想到一般上传的后们文件是php，存在一个sql注入的过程，先筛选一下http contains”and”，可以看到存在了一些注入的数据。

继续追踪一下，我们可以看到，使用sqlmap的痕迹，我们将特征进行追踪一下，http contains”sqlmap/1.0-stable”

这么多的sql注入的包，但是肯定存在一个最关键的包可以拿到webshall，那最后找到是20160809092842.php
